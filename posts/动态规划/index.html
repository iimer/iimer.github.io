1 理论

1.1 引入

每当根据题目遍历过程过，不能根据已有数据集，得到当前下标预期的值的时候，就可以考虑是否可以使用动态规划。

如果根据题目可以提取出一个状态转移方程，那么此题便是可以用动态规划解。

1.2 状态转移方程定义

状态转移方程可以是根据上一个状态,也可以是根据前面部分或全部的状态得到当前的状态。

1.3 分类

根据我自己的刷题经验，动态规划题可以分为以下几类：

  • 单串问题：题目比较容易看出状态转移方程并且使用一维数组。
  • 双串问题（包括背包问题）：状态转移方程有的较为难看出，需要使用二维数组，临界问题需要着重解决。
  • 复杂题型：需要穿插深搜等知识，配合动态规划来解决问题，比较灵活，状态转移方程只是解题的一环，一般不是最终答案。

1.4 技巧

一定需要用到数组吗？

不一定。

不用数组的情况：状态转移方程中当前状态只需要根据上一个状态就可以得到。

个人倾向于先用数组来解题，以此来减少题目的复杂性，并在解决完之后，进行代码层面的优化，即可消除数组。

2.2 双串问题

97. 交错字符串

给定三个字符串 s1、s2、s3，请你帮忙验证 s3 是否是由 s1 和 s2 交错 组成的。

两个字符串 s 和 t 交错 的定义与过程如下，其中每个字符串都会被分割成若干 非空 子字符串：

  • s = s1 + s2 + ... + sn
  • t = t1 + t2 + ... + tm
  • |n - m| <= 1
  • 交错 是 s1 + t1 + s2 + t2 + s3 + t3 + ... 或者 t1 + s1 + t2 + s2 + t3 + s3 + ...

**注意：**a + b 意味着字符串 a 和 b 连接。

  • 0 <= s1.length, s2.length <= 100
  • 0 <= s3.length <= 200
  • s1、s2、和 s3 都由小写英文字母组成

分析：

排除双指针：在匹配到s3一个字符的时候，假如s1、s2的当前位置都是那个字符，则不好进行下去。

排除一维数组dp：dp[i]代表i下标的s3判断符合题意，在dp[i-1]符合题意的情况下，无法确认下一个字符在s1或s2中的位置。

二维数组dp：

假定k,i,j从1开始代表各个的位置且k=i+j，k位置的s3想要满足题意的话，则i位置的s1的值或者j位置的s2的值等于s3[k]，且对应的k-1与i-1和j，或与i和j-1是满足题意的。

定义boolean类型的二维数组，则得到状态转移方程：dp[i][j] = dp[i-1][j]&&s1a[i-1]==s3a[i+j-1] || dp[i][j-1]&&s2a[j-1]==s3a[i+j-1];。

考虑到临界，dp数组需要多出一行一列来记录只取一边的情况，且第一行和第一列需要单独初始化来避免状态转移方程的数组越界问题，且dp[0][0]为true。

此种情况下，s1代表dp的行下标，s2代表dp的列下标，需要着重注意，不可混淆。

题解：

public boolean isInterleave(String s1, String s2, String s3) {
        if (s1.length()+s2.length()!=s3.length()) {
            return false;
        }
        boolean[][] dp = new boolean[s1.length()+1][s2.length()+1];
        dp[0][0] = true;
        char[] s1a = s1.toCharArray();
        char[] s2a = s2.toCharArray();
        char[] s3a = s3.toCharArray();
        for (int i = 1; i <= s1a.length && s1a[i-1]==s3a[i-1]; i++) {
            dp[i][0] = true;
        }
        for (int i = 1; i <= s2a.length && s2a[i-1]==s3a[i-1]; i++) {
            dp[0][i] = true;
        }
        for (int i = 1; i <= s1a.length; i++) {
            for (int j = 1; j <= s2a.length; j++) {
                dp[i][j] = dp[i-1][j]&&s1a[i-1]==s3a[i+j-1] || dp[i][j-1]&&s2a[j-1]==s3a[i+j-1];
            }
        }
        return dp[s1.length()][s2.length()];
    }

题后感：

根据题目进行的分析，较为复杂，容易出错。

二维数组dp的特性是根据二维数组来解题，不做变型的反映到题目应如：62.不同路径
